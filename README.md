## 🚀 Deploy to GitHub Pages (free hosting)

You'll have a live URL like `https://YOUR-USERNAME.github.io/portfolio/` in about 5 minutes.

### Option A — User site (recommended for a personal portfolio)

This gives you the cleaner URL `https://YOUR-USERNAME.github.io/`.

1. **Create a new repo** on GitHub named exactly: `YOUR-USERNAME.github.io`
   (replace `YOUR-USERNAME` with your actual GitHub handle, e.g. `AhmedBassoul1.github.io`)
2. **Upload the files** (`index.html`, `styles.css`, `script.js`) to the repo's root.
   Either drag-and-drop in the GitHub web UI, or via terminal:
   ```bash
   git init
   git add .
   git commit -m "Initial portfolio"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/YOUR-USERNAME.github.io.git
   git push -u origin main
   ```
3. Go to the repo → **Settings** → **Pages** → under "Build and deployment", set:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / `(root)` → Save
4. Wait ~1 minute. Your site is live at `https://YOUR-USERNAME.github.io/`

### Option B — Project site

If you'd rather host it under an existing account:

1. Create any repo (e.g. `portfolio`)
2. Push the files to `main`
3. Settings → Pages → Source = `main` / `(root)`
4. Live at `https://YOUR-USERNAME.github.io/portfolio/`

Before deploying, do a quick pass on `index.html`:

- [ ] Update LinkedIn URL (currently `linkedin.com/in/ahmed-bassoul`)
- [ ] Update GitHub URL (currently `github.com/AhmedBassoul1`)

- [ ] Optional: add a profile photo or project screenshots
- [ ] Optional: update Open Graph meta tags with a real `og:image`

To change the **accent color**, edit one line in `styles.css`:

```css
:root {
  --accent: #c9304a;  /* change this */
}
```
